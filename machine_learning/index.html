Machine Learning

Machine Learning Mastery A Tour of The Most Popular Machine Learning Algorithms

常见机器学习算法及代码 categorized by learning styles and similarity

  • Supervised Learning: Input data is called training data and has a known label or result.
    • Regression (discrepancy)
      • Ordinary Least Squares Regression
      • Logistic Regression
      • Multivariate Adaptive Regression Splines (MARS)
      • Locally Estimated Scatterplot Smoothing (LOESS)
    • Classification (confusion matrix, accuracy, AUC, information gain, F-score)
      • KNN
      • Back Propagation Nerual Network
  • Unsupervised Learning: Input data is not labeled and does not have a known result.
    • Clustering
      • K-Means
      • Hierachical Clustering
    • Dimensionality Reduction
      • Factor Analysis
      • Principle Component Analysis (PCA)
    • Association
      • Apriori Algorithm
    • K-Nearest Neighbour (KNN)
  • Semi-Supervised Learning: Input data is a mixture of labeled and unlabelled examples.
  • Reinforcement Learning 需要的数据量少，甚至无需数据，但训练资源消耗很大，陷入局部最优解
    • 不完全信息对弈决策（德州扑克）deepstack
    • 随机性动态规划
    • 完全信息对弈决策（围棋）

深度网络：前馈网络、递归网络、自组织网络 存在问题：依赖数据与标注，逻辑不足

知识推理，优化决策

AutoML自动搜索最优的网络结构和参数，元学习

未来：（类脑）新构架，胶囊网络，即插即用模块
