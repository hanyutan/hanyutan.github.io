Scoop

Scoop，优质的 Windows 包管理器，一行代码搞定安装和卸载，干净又迅速。

Install Scoop and Apps

Windows10在开始菜单键右击选择Windows PowerShell，依次输入下列命令：

# Windows PowerShell命令行输入
set-executionpolicy remotesigned -scope currentuser	
$env:SCOOP='D:\Scoop'      # 替换到你自定义的安装目录，以后安装的其他软件也会安装到此目录下
[environment]::setEnvironmentVariable('SCOOP',$env:SCOOP,'User')
iex (new-object net.webclient).downloadstring('https://get.scoop.sh')

# 一行代码即可安装和卸载
scoop install <app name>         # 安装软件，D:\Scoop\apps出现对应的文件夹
scoop uninstall <app name>       # 卸载超简单
scoop uninstall scoop                 # 也可以用此功能卸载Scoop自己

Scoop 默认软件仓库（main bucket）包含了cuda、git、go、hugo、julia、kafka、latex、mysql、nano、nodejs、octave、pandoc、perl、php、r、ruby、scalaspark、sudo、

但main bucket的软件数量是有限的，所以需要添加其他bucket（软件仓库）来下载我们想要的软件的安装配置文件。在这里可以直接搜索想要的软件，找到对应包含它的bucket。

举例：

  • 官方维护的bucket
    • 安装命令scoop bucket add extras，包含了anaconda3、firefox、foxit-reader、notepadplusplus、rstudio、sublime-text等
    • 安装命令scoop bucket add java，包含了oraclejdk12等
    • 安装命令scoop bucket add JetBrains，包含了IntelliJ-IDEA等
  • 社区提供的bucket——dorado ，安装命令scoop bucket add dorado https://github.com/h404bi/dorado，包含了wechat等。
  • 指定安装某个bucket下的某项app：scoop install <bucket name>/<app name>

补充：What’s bucket?

Scoop 将一个个bucket（仓库）缓存至本地，当我们想要安装一个软件的时候，Scoop 就从本地的仓库中挑选出我们想要安装的软件的安装配置文件，并依照这个配置文件进行软件的安装工作，包括：

  • 从哪里下载软件
  • 如何安装软件、安装到哪里、需要修改更新什么环境变量
  • 安装之前、之后都要做什么准备（善后）工作

查看一个软件的安装配置详情，scoop info <app name>:

# 以hugo为例，其中 Manifest 项目下的路径指向的文件是Scoop具体读取的配置文件，我们的bucket就是由这样的一个个配置文件组成的。

Name: hugo
Description: Speedy and flexible static site generator (without Sass/SCSS support)
Version: 0.58.1
Website: https://gohugo.io
License: Apache-2.0 (https://spdx.org/licenses/Apache-2.0.html)
Manifest:
  D:\#Han Yutan\Scoop\buckets\main\bucket\hugo.json
Installed:
  D:\#Han Yutan\Scoop\apps\hugo\0.58.1
Binaries:
   hugo.exe

Create my own bucket

在gitee中创建新仓库并克隆到本地，向文件夹中添加创建app的安装配置文件(.json)，最后将本地更改推送至gitee。

如何对json文件进行配置？请参考官方说明 App Manifests

添加bucket97代码：scoop bucket add bucket97 https://gitee.com/hanyutan/bucket97，安装app代码scoop install bucket97/<app name>.
