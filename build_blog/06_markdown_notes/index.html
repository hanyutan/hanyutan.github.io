Code and Syntax Highlighting

单行代码使用1个反引号\`将内容两边括起来，例如：`code`。
代码块使用3个反引号\```将内容括起来（单独占一行）。在第一行末尾加上所使用的语言将会按照该语言的规则进行highlight。

常用的有bash、c++、c#、css、django、go、http、java、javascript、json、julia、markdown、mathematica、matlab、perl、python、r、ruby、sas、shell、sql、stata、tex、html、xml等。

全部可支持的语言列表请参考<https://highlightjs.org/static/demo/>
