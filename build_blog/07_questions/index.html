Q&A

1.如何在 Github Pages 中输入数学公式？

<script type="text/javascript"
src="http://cdn.mathjax.org/mathjax/latest/MathJax.js?config=TeX-AMS-MML_HTMLorMML">
</script>

# markdown内容
使用2个dollar符号括起来可以让公式单独显示一行$$\prod(1+1)$$
写作markdown文件时发现用$$标记的公式内容无法在网页上正常解析成数学公式，只能显示原有内容。

解决办法：使用 Mathjax 渲染数学公式。 需要找到你使用的Hugo Themes文件夹里的<your site>\themes\<theme name>\layouts\_default\baseof.html文件， 打开后在<head>里加入右侧代码。

2.如何开启双语界面？

# 原有内容，说明网站现在是英文的
[languages.en]
  languageCode = "en-us"
# 在其后面添加中文
[languages.zh]
  languageCode = "zh-cn"
# 回到config.toml开头修改默认语言
defaultContentLanguage = "zh"
defaultContentLanguageInSubdir = true

解决办法：在config.toml文件里找到[languages]部分，按照右侧代码修改。 然后在<your site>\themes\<theme name>\i18n中保留en.yaml和zh.yaml文件，并把其中翻译不当的地方修改一下。 之后新增博客时如果要增加英文版本，只需在文件扩展名前加.en即可。例如中文版本的为my_post.md，那么英文版本为my_post.en.md。

4.如何让我的github.io被Google/Bing/Baidu搜索到？

在Google搜索页面中输入`site:.github.io`发现这个网页无法被Google索引到。

  • Google: 根据搜索结果页面的提示，使用Google Search Console （需要拥有Google账户）， 选择资源类型————网址前缀，输入完整的urlhttps://<username>.github.io，下载HTML验证文件上传到github.io的repository中，等待官方验证通过即可。

5.如何对我的github.io进行网站数据统计与分析？

解决办法：

  • 百度统计：注册并登陆百度统计 ，添加网站后， 在左边栏的代码管理————代码获取中复制<script>...</script>这一大段添加到你的 themes/<theme_name>/layouts/_default/baseof.html文件中的部分。 如果代码安装正确，一般20分钟后，可以查看网站分析数据。

6.如何在Markdown里输入空格以及换行？

解决办法：利用HTML语言 原本内容即可在内容前增加一个空格，在尾部添加
标签即可换行。 此外，在尾部敲入几个空格也可实现markdown里的换行，留一个空行可以实现另起一段。

7.Github Pages可以使用多个主题吗？

解决办法：同一个repo下应该只能使用一个主题，但是可以新建其他repo，下设gh-pages分支， 然后另设一个New Site把public文件夹里的内容push到gh-pages分支上，作为项目主页， 同时单独的分支又可以不影响master上主要项目的开发。

8.没有Vpn访问Github太慢，或者担心所在地区某一天被美国封锁而无法使用Github怎么办？

# 在原来链接到Githuhb的本次仓库中右击Git Bash，重新管理远程链接
git remote rm origin
git remote add github https://github.com/username/username.github.io  # 添加远程分支，命名github
git remote add gitee https://gitee.com/username/username              # 添加远程分支，命名gitee

# push到github
git push -u github master

# push到gitee
git push -u gitee master

解决办法：那就支持一下国产吧！
来段摘自官方的介绍：“开源中国是目前国内最大的开源技术社区，拥有超过200万会员， 形成了由开源软件库、代码分享、资讯、协作翻译、讨论区和博客等几大频道内容， 为IT开发者提供了一个发现、使用、并交流开源技术的平台。”

官方网站码云Gitee：https://gitee.com

Gitee有自己的Page，和Github操作几乎一模一样，而且支持从Github上直接克隆Repo以及同步更新。 如何建立码云Pages请参考：http://git.mydoc.io/?t=154714#text_154714，注意在建仓、push后还需要启动Gitee Pages服务。

想自己本地上传同时push两个remote的话，可以参考代码栏。

补充
我发现这种做法会让https://username.gitee.io或者https://username.oschina.io的链接直接指向https://username.github.io，那访问速度相当于没变化啊！ 猜测是因为网页配置文件config.toml里的url没有改，所以还是不要关联到同一个本地仓库，分开比较好。 还有网站流量统计、被各大搜索引擎搜索到的验证文件也需要重新弄一遍。
