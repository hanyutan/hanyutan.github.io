Tips of Jupyter Notebook

  1. 如何在任意位置打开jupyter notebook？

按住Shift+右键点击“在此处打开命令行”/“在此处打开powershell窗口”，输入jupyter notebook即可以当前路径为根目录打开jupyter notebook。
关于win10右键菜单没有“在此处打开命令行”选项的处理，请参考本博客Build Blog - Q&A的第9条。

  1. 如何在jupyter notebook里执行shell命令

    !pip list   # 直接在shell命令前加叹号即可

  2. Jupyter Notebook的一些有用拓展 link

安装 nbextensions 插件，勾选Hinterland、Snippets、Split Cells Notebook、Table of Contents

pip install jupyter_contrib_nbextensions
jupyter contrib nbextension install --user
jupyter nbextension enable codefolding/main
