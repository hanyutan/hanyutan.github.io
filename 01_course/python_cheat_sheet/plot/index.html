Plot

  • matplotlib 数学绘图库 可视化工具 参数较复杂
  • seaborn 在matplotlib的基础上进行更高级的API封装 官网
  • yellowbrick
  • pyecharts 用于生成Echarts图表的类库 官方教程 link
  • import matplotlib.pyplot as plt
plt.style.use('ggplot')    # 将风格设置为近似R中的ggplot库
%matplotlib inline 
import matplotlib as mpl
mpl.rcParams['font.sans-serif'] = ['SimHei']   # 显示中文
mpl.rcParams['axes.unicode_minus'] = False
import pandas as pd
pd.set_option('display.float_format',lambda x:'%.4f' % x)#不用科学计数法

import seaborn as sns
sns.set_style('whitegrid')

from matplotlib import animation  # 动画
%matplotlib notebook       # 在notebook内实现matplotlib的动画、立体显示效果

import warnings
warnings.filterwarnings('ignore') #为了整洁，去除弹出的warnings

  • 设置绘图窗口尺寸

    • fig = plt.figure(dpi=,figsize=(,))
    • fig,ax = plt.subplots(2,2,figsize=(6,4)) # subplots is iterable, ax可用
    • dpi屏幕分辨率，默认为80
    • figsize元组指定图表的宽度、高度

  • 折线图 plt.plot(x,y,linewidth=,alpha=,label=)

    • 组合设置’r.’ 颜色+形状 colors = r/b/y/g/k
      markerSymbols = ./*/o/v/^/</>/s/p/P/-/–
    • linewidth设置线条粗细
    • alpha设置线条透明度
    • plt.fill_between(x,y_high,y_low,facecolor=,alpha=)填充区域颜色及透明度

  • 散点图 plt.scatter(x,y,c=,edgecolor=,s=,marker=,label=)

    • c/color设置数据点颜色
    • edgecolor设置数据点轮廓颜色 ‘none’默认
    • marker设置数据点形状 o * + v
    • s设置点的尺寸

  • 直方图 plt.hist(x,bins=,color=,label=) ax.hist()

    • bins
    • width

  • 条形图/柱状图 plt.bar(x,height,width=,bottom=,color=,label=,yerr=,align=) ax.bar()

    • x不是直接的分类标签，而是柱子的横坐标位置
    • 一般用info=df[“var”].value_counts()得到降序的属性值计数，然后x=range(len(info))，height=info，names=list(info.index)
    • 设置横坐标标签为分类标签plt.xticks(x,names)
    • width条形宽度
    • bottom的最大值设置为y数据条形的最小值
    • yerr标准差
    • align条形在x刻度中间or边缘 ‘center’ ‘edge’

  • 横向条形图

  • 饼图 plt.pie(x, labels=, explode=, startangle=90, autopct=“%1.1f%%” )

    • x是计数数据，lables是分类标签名称
    • explode为一个数值列表，可以控制某一个扇形块分离的距离[0.1,0,0,0,0]
    • startangle=90饼图从正上方开始逆时针绘制
    • autopct=“%1.1f%%“在扇形块显示所占百分比，保留一位小数

  • 密度分布图 (density) sns.kdeplot(x, shade=True, color=)

  • 箱线图 df[“var”].to_frame().boxplot(widths=0.5,fontsize=14)

  • 小提琴图

  • 热力图 heatmap

  • 等值线图 contour map

  • 雷达图 radar

  • 气泡图

  • 子图

    • plt.subplots(m,n,figsize=(wigth,height))
    • plt.subplot(m,n,i) # 绘制在第几张子图上

  • 图例

    • plt.legend([name_list],loc=)
    • 可以在每个绘图语句中家label=，或者在最后统一指定[name_list]，但顺序可能出错
    • 在plt.plot()中用参数label分别指定内容，然后结尾处添加plt.legend(loc=)指定位置，从右上角开始逆时针分别为1 2 3 4，默认为1

  • 图表标题

    • plt.title(“内容”,fontsize=,y=) ax.set_title()
    • fontsize字号
    • y调整标题在y轴方向相对于图片顶部的位置
    • plt.suptitle() / fig.suptitle()为所有子图添加总标题

  • 坐标轴名称

    • plt.xlabel(“内容”,fontsize=) plt.ylabel(“内容”,fontsize=)
    • ax.set_ylabel(“label”,fontsize=) ax.set_xlabel(“label”,fontsize=)

  • 坐标轴刻度及范围

    • plt.axis([x_min,x_max,y_min,y_max]) # 设置刻度范围
    • plt.xlim(0,1) plt.ylim(0,1)
    • 调用fig.autofmt_xdate()来绘制斜的日期标签，以免彼此重叠
    • plt.xticks([location],[labels],rotation=,fontsize=) # 设置x刻度位置及内容
    • ax.set_xlim(x_min,x_max)
    • ax.set_xticks([location]) # 设置x刻度位置
    • ax.set_xticklabels([labels],rotation=,fontsize=) # 设置刻度标签内容、旋转角度
    • for i in ax.flat: plt.setp(i.get_xticklabels(),rotation=-90) # 对坐标刻度标签的后续调整，ax.flat()方法遍历子图的每个轴，并使用.setp()设置特定的属性
    • plt.tick_params(axis=,which=,labelsize=) # axis设定影响哪条轴上的刻度 ‘x’ ‘y’ ‘both’；which ‘major’ ‘minor’ ‘both’；labelsize刻度标记的字号

  • 隐藏坐标轴 plt.axes().get_xaxis().set_visible(False) plt.axes().get_yaxis().set_visible(False)

  • 坐标轴网格 plt.grid(axis=“y”,color=“k”)

  • 显示图片

    • plt.show()
    • plt.tight_layout() fig.tight_layout() # 自动调整子插图，避免排版上显得过于拥挤

  • 保存图表

    • plt.savefig(filepath,bbox_inches=)
    • filepath指定文件保存路径及名称，jpg/png格式
    • bbox_inches=‘tight’将图表多余的空白区域剪裁掉

颜色设置

  • c1, c2 = sns.color_palette(‘Set1’, 2)
  • 颜色名称 ‘blue’
  • 自定义RGB——元组，三个0-1的小数值对应红绿蓝的分量，越接近1颜色越浅 (0,0,0.99)

  • 颜色映射 colormap是一系列颜色，从起始颜色渐变到结束颜色（plt.scatter）

    • c=y 将参数c设置为一个y值列表，并使用参数cmap指定使用哪个颜色映射
    • 更多请访问http://matplotlib.org/ ，点击Examples，向下滚动到Color Examples，再单击colormaps_reference

    • https://matplotlib.org/gallery/color/colormap_reference.html#sphx-glr-gallery-color-colormap-reference-py

      # 3D图像
from mpl_toolkits.mplot3d import axes3d
import matplotlib.pyplot as plt
from matplotlib import cm
fig = plt.figure()
ax = fig.gca(projection='3d')
X, Y, Z = axes3d.get_test_data(0.05)
ax.plot_surface(X, Y, Z, rstride=8, cstride=8, alpha=0.3)
cset = ax.contour(X, Y, Z, zdir='z', offset=-100, cmap=cm.coolwarm)
cset = ax.contour(X, Y, Z, zdir='x', offset=-40, cmap=cm.coolwarm)
cset = ax.contour(X, Y, Z, zdir='y', offset=40, cmap=cm.coolwarm)
ax.set_xlabel('X')
ax.set_xlim(-40, 40)
ax.set_ylabel('Y')
ax.set_ylim(-40, 40)
ax.set_zlabel('Z')
ax.set_zlim(-100, 100)
plt.show()
