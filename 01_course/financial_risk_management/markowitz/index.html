Breaking Down Finance

Modern portfolio theory

Modern portfolio theory (MPT) 证券投资组合理论 is a method for constructing a portfolio of securities

Precondition:

  • investors only care about the mean return and the standard deviation of securities

Capital allocation line

资本配置线 Capital allocation line (CAL) or Capital market line (CML) is about how much to invest in the risky asset (eg. shares) and riskfree asset (eg. government bond)

  • Riskfree asset: standard deviation is zero
  • Risky asset: higher expected return compensating for risk

Constructing portfolios: combine both assets into a portfolio
Expected return of the portfolio $r_p$: ($w_r$ is fraction)

$$E(r_p)=E(r_r)*w_r+(1-w_r)*E(r_f)$$

Riskiness of the portfolio $r_p$:

$$\sigma_p=w_r*\sigma_r$$

CAL contains all different combinations of both assets.
Since there is linear relation ship between $E(r_p)$ and $\sigma_p$, the capital allocation line is a straight line.

The slope of CAL: $slope=\frac{E(r_r)-r_f}{\sigma_p}$ measures the trade-off between return and risk, that is to say how much expected return investors will receive in exchange for taking risk.

Sharpe ratio: risk/return trade-off
Higher sharpe ratio of risky portfolio leads to steeper CAL. And when we determine the composition of the risky portfolio, we should let the sharpe ratio as high as possible, for steeper CAL compensate more for taking additional risk.

Markowitz’s Portfolio Selection Model

均值-方差投资组合理论(MPT)，求解：Lagrange Multiplier
about how to allocate the money to construct a portfolio that gives investors the best risk/return trade-off available (minimize risk given expected return, or maximize return given risk)

Markowitz投资组合模型：
以预期收益率度量收益，以收益率方差度量风险
教科书中一般以历史收益率均值作为未来期望收益率（造成“追涨”效果），以历史收益率的协方差度量资产风险与相关性
根据每种证券的预期收益率、方差、证券之间的协方差矩阵，计算得到投资组合的有效边界
再根据投资者的效用无差异曲线，确定最佳投资组合

模型前提：

  • （1）投资者希望财富越多越好，且被投资效用为财富的增函数，但财富的边际效用是递减的。
  • （2）假定收益率正态分布
  • （3）投资者希望投资效用的期望值最大而该期望值是预期报酬率和风险的函数，因此影响投资决策的主要因素是预期报酬率和风险。
  • （4）投资者对风险是反感的，投资风险以预期报酬率的方差或标准差来表示。
  • （5）投资者是理性的。$他遵循的原则是$：在相同的预期报酬率下选择风险小的证券，或者在相同的投资风险下选择预期报酬率最大的证券。
  • （6）市场的有效性，即对本市场上一切信息都是已知者

$Return = \frac{x_2-x_1}{x_1}$ or $Return = log(\frac{x_2}{x_1})$

Expected return of the portfolio $r_p$: ($w_A, w_B$ are fraction)

$$E(r_p)=w_A*E(r_A)+w_B*E(r_B)$$

The riskiness of the portfolio $r_p$:

$$\sigma_p^2=w_A^2\sigma_A^2+w_B^2\sigma_B^2+2w_Aw_Bcov(r_A,r_B)$$

  • $cov(r_A, rB) = \rho{A,B}\sigma_A\sigma_B$
  • securities are potentially correlated
  • $\rho$ is the correltation [-1,1], $\rho=-1$ risk is lowest, $\rho=1$ risk is highest.

Two assets portfolio (different $\rho$)

# load packages
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
plt.style.use('ggplot')           # set style as ggplot2 in R
%matplotlib inline
import seaborn as sns
import warnings
warnings.filterwarnings('ignore')

# set parameters
r1=0.3;r2=0.25
sigma1=np.sqrt(0.1);sigma2=np.sqrt(0.05)

# draw portfolios for different rho
rho_list=[-1,-0.5,0,0.5,1]
color_list = sns.color_palette('Set1', 5)
for i,rho in enumerate(rho_list):
    df = pd.DataFrame({"a":np.linspace(0,1,100)})
    df["Rbar"] = df["a"].apply(lambda x: (1-x)*r1+x*r2)
    df["sigma"] = df["a"].apply(lambda x: np.sqrt((1-x)**2*sigma1**2+x**2*sigma2**2+2*rho*x*(1-x)*sigma1*sigma2))
    plt.plot(df["sigma"],df["Rbar"],color=color_list[i],marker='.',label="rho="+str(rho))
plt.legend(loc=2)
plt.show()

Effiecient frontier:
$\rho$ is given and weights are varying, plot the relationship between return and riskiness for all possible combinations
the line is called “minimum variance frontier”, indicates the minimum riskiness that can be reacheed for each level of expected return
efficient frontier is part of minimum variance frontier, higher return for the same risk (仅最左边点的右上边部分)

给定预期收益率，选择最小风险；给定风险，选择最高的预期收益率

All the portfolios on this frontier provides the best return for a given amount of risk. Above the line cannot be reached, below the line are inefficient.

Frontier is non-linear and its second derivative < 0.

Portfolio optimization

determine the optimal risky portfolio

Mean-Variance Optimization:
Select the portfolio on the efficient frontier with the highest sharpe ratio, which means that the CAL is tangent to efficient frontier. This particular portfolio servers as the risky asset, with riskless asset they construct the investor’s overall portfolio.

how to determine weights depends on how much risk the investor is willing to take.

Limitations of portfolio optimization

  • estimate lots of variables: suppose there are n securities, we need to estimate n expected returns + n variances + n(n-1)/2 correlation coefficients.
  • sensitive to outputs when (expected returns / standard deviation / correlations) change slightly
  • often obtain extreme weights (suggest to allocate everything in just a hand full of shares)

Factor models

relate the return of a security to a number of (risk) factors

$$r_{i,t}=\alpha_i+\beta1F{1,t}+\beta2F{2,t}+…+\betanF{n,t}+\epsilon_{i,t}$$

where:

  • $r_{i,t}$ is security $i$’s return at time $t$
  • $F{1,t},…,F{n,t}$ are factors (eg. macroeconomic variables, stock indices, or other portfolios)
  • $\beta_1,…,\beta_n$ are factor loadings which measure the sensitivity of the security to changes in the factor
  • $\epsilon$ is called error term, represents security-specific news that becomes available to investors
  • $\alpha$ is refered to as “security’s alpha” (most interested in), measures the security’s excess return, which cannot be explained by those factors and error term

objectives to achieve when construct portfolio:

  • maximize the (excess) return –> maximize $\alpha$
  • minimize volatility (stock market fluctuations) –> minimize $\beta$
  • minimize the impact of firm-specific news –> being sufficiently diversified

Note: the approach only make sense if the investors can correctly determine which stocks have a positive $\alpha$, thus investors should have stock-picking skills

Treynor-Black model can solve the above problem

Treynor-Black model

a type of active portfolio management

积极的资产组合管理： - 相信市场非有效，寻找错误定价的股票，以获得正的超额收益 - 构造一个sharpe ratio最大的portfolio - 包含两种模式：时机选择（宏观经济因素），证券选择（微观经济因素） - 流程：发现超额收益股票，配置超额收益股票以构建active portfolio，引入市场组合最大化sharpe ratio来构建optimal portfolio，引入riskfree asset并结合个人风险态度来确定最终资产配置 消极的资产组合管理： - 认为市场完全有效 - 证券选择毫无意义，只需持有市场指数资产组合与货币市场基金（安全资产）

Optimal risky portfolio consists of a passive (market) portfolio and an ative portfolio for which we have alpha forecasts. These forecasts are obtained using a factor model.

1.determine weights

  • Treynor-Black ratio / appraisal ratio: $\frac{\alpha_i}{\sigma(\epsilon_i)^2}$,
    where $\sigma(\epsilon_i)^2$ is volatility unrelated to factors(eg. unsystematic risk),
    it measures the value the security would add to our portfolio, on a risk-adjusted basis
  • weights should be proportional to Treynor-Black ratio
  • security with higher alpha and lower volatility, has higher weight

Optimal weights:

$$w_i=\frac{\frac{\alpha_i}{\sigma(\epsiloni)^2}}{\sum{j=1}^n\frac{\alpha_j}{\sigma(\epsilon_j)^2}}$$

Construct the active portfolio:

  • $\alpha_A=\sum _{i=1}^Nw_i\alpha_i$
  • $\beta_A=\sum _{i=1}^Nw_i\beta_i$
  • $\sigma(\epsilon_A)^2=\sum _{i=1}^Nw_i^2\sigma(\epsilon_i)^2$

CAPM

资本资产定价模型，给风险定价，认为只有systematic risk才能带来收益，因为nonsystematic risk可以通过投资组合的方式消除掉

期望收益率：$E(\bar{r}_i)-r_f=\beta_i[E(\bar{r}_m)-r_f]$
实际收益率：$\bar{r}_i-r_f=\beta_i[\bar{r}_m-r_f]+e_i$ 或者 $\bar{r}_i=\beta_i\bar{r}_m+(1-\beta_i)r_f+e_i$

  • $\bar{r}_i$为目标portfolio的收益率，$r_f$为risk-free asset的收益率，比如国债的等固收资产
  • $\bar{r}_m$为市场投资组合(market portfolio)的收益率，衡量整个市场所有风险的收益情况
    可以用大盘指数（HS300）代替 以它为基准来评估$\bar{r}_i$的提升量
  • $E(\bar{r}_i)-r_f$称为“超额收益”or“风险溢酬”
  • 公式表示portfolio的风险收益只和市场的风险收益挂钩， 可以写成协方差形式再代入：$\sigma _{im}=Cov(\bar{r}_i,\bar{r}_m)=Cov(\beta_i\bar{r}_m+(1-\beta_i)r_f+e_i,\bar{r}_m)$ $=Cov(\beta_i\bar{r}_m, \bar{r}_m)=\beta_i\sigma_m^2$从而得到$\beta_i=\frac{\sigma _{im}}{\sigma_m^2}$
  • 当大盘风险溢筹为正时，$\beta$越大收益越大；当大盘风险溢筹为负时，$\beta$越大亏损越大。 在实际投资过程中，机构通常用对冲的方式抵消大盘带来的systematic risk，从而使portfolio跑赢大盘。
  • $\beta$系数是portfolio的beta值，度量了portfolio系统风险的相对大小，即和市场相关的风险
    • $|\beta|>1$代表portfolio比大盘波动更剧烈
    • $|\beta|<1$代表portfolio波动小于大盘
    • $\beta <0$说明portfolio波动方向与大盘相反
    • $\beta_i=\frac{\sigma _{im}}{\sigma_m^2}$刚好也是上述方程的回归系数，两个beta定义等价

Jensen在实际应用中引入$\alpha$参数，$\bar{r}_i-r_f=\alpha_i+\beta_i[\bar{r}_m-r_f]+e_i$， 这里的收益率分为系统风险收益（$\beta$收益）和超额收益（$\alpha$收益），可以使用最小二乘法直接求解。 - $\alpha$代表收益率胜过大盘的部分，用于衡量选股能力，并作为基金经理人的绩效指标

  • 选股目标是找到最大的$\alpha$，给定收益率，$\alpha$越大则$\beta$越小
  • $\alpha$的pvalue<0.05，显著胜过大盘，除了大盘波动带来的收益外，其自身价值产生了$100\alpha%$的收益
  • $\beta$的pvalue<0.05，显著，假如大盘涨了$10%$，则portfolio预期涨$10\beta%$
  • R方看拟合效果

名词解释：

  • Utility效用：消费者满足的度量，福利指标
  • 边际效用：增加一个单位的商品或服务，给消费者带来的新增效用。（效用函数的一阶导数）
  • 边际效用递减原理：效用函数一阶导数>0（越多越好），二阶导数<0（新增效用变少）。
  • 无差异曲线：每一点对应的商品组合不同，但得到的效用相同。高风险高期望，低风险低期望。
