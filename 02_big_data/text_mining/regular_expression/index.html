Regular Expression

正则表达式 Regular Expression

正则表达式是用于处理字符串的强大工具

Metacharacter Description
^ 开头位置，[ ]内代表“非”
$ 结尾位置
. 匹配任意字符（除换行符”\n”外），[ ]内代表句点
[ ] 字符集，匹配方括号内任意字符
逐个列出or给出范围 [a-z] [x-z] [0-9]
( ) 分组，视作整体 can be recall later? \n匹配第n个subexpression
* 前一个元素出现任意次数 >=0
{m,n} 前一个元素出现 m<=次数<=n，{m}，{m,}

想输入特殊符号本身就用转义字符\

预定义字符集:

[^\d]

字符 含义 字符 含义
\d [0-9] 数字 \D [^\d] 非数字
\s [<空格>\t\r\n\f\v] 空白字符 \S [^\s] 非空白字符
\w [A-Za-z0-9] 单词字符 \W 非单词字符

向量空间模型

给定文档集D，每一篇文档都是一个词集，目标是用向量$d_i$表示文档：

$$d_i=(w _{1,i},w _{2,i},…,w _{N,i})$$

其中$w_{1,i}$表示$document_i$中$word_j$的权重，而N是词语的数量。

One-hot方法：
$w_{j,i} = 1$ if $word_j\in document_i$

$w_{j,i} = 0$ if $word_j\notin document_i$

TF-IDF方法：
用“词频-逆文本频率”表示权重，如果一个词在一篇文档中出现频率较高，而在其他文档中出现频率较低，则获得的权重更高。 避免了一些常用的助词、停用词被赋予高权重。

$$w _{j,i}=tf _{j,i}\times idf _{j}$$

  • $tf_{j,i}$表示$word_j$在$document_i$中出现的频率
  • $idf_{j}$是$word_j$在全文档集D上的逆文本频率，用所有文档数量除以包含$word_j$的文档数量再取对数。

$$idf_j=log_2\frac{|D|}{ | \lbrace document\in D | j\in document \rbrace | }$$
