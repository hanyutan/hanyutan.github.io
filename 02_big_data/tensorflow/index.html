Reference: 《Tensorflow实战》 黄文坚 唐源 电子工业出版社

Tensorflow Introduction

Github网址：https://github.com/tensorflow/tensorflow/

To install the current release for CPU-only:$ pip install tensorflow
Use the GPU package for CUDA-enabled GPU cards:$ pip install tensorflow-gpu

Tensorflow使用数据流式图来规划计算流程：

  • 将计算表示为一个有向图(directed graph)，或称计算图(computation graph)
  • 节点(node)是运算操作(operation)的实例化
  • 节点与节点之间的连接称为边(edge)
  • 在计算图(computation graph)的边(edge)中流动(flow)的数据被称为张量(tensor)
    其数据类型可以事先定义，也可以根据计算图的结构推断得到
  • 运算核(kernal)是一个运算操作在某个具体硬件(e.g.: CPU/GPU)的实现
  • Session是用户使用Tensorflow时的交互接口，通过extend方法添加新的节点和边来创建计算图，然后通过run方法执行计算图
  • 计算图(computation graph)通常会被创建一次，然后执行很多次，数据张量(tensor)只在图中流过一边而不会被保留
  • Variable是一类特殊的运算操作，可以将一些需要保留的tensor（比如神经网络的系数）储存在内存或者显存中，在训练时也可以更新并保存，
    从而可以实现assign/assignadd(+=)/assignmul(*=)等特殊操作
类型 内建运算操作
标量运算 add, sub, mul, div, exp, log, greater, less, equal
向量运算 concat, slice, split, constant, rank, shape, shuffle
矩阵运算 matmul, matrixinverse逆, matrixdeterminant行列式
带状态的运算 variable, assign, assignadd(+=), assignmul(*=)
神经网络组件 softmax, sigmoid, relu, convolution2d, maxpooling
储存、恢复 save, restore
队列及同步运算 enqueue, dequeue, mutexacquire, mutexrelease
控制流 merge, switch, enter, leave, nextiteration
